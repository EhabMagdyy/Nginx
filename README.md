## 1. Nginx as Web Server
### Files locations (in ubuntu)
```bash
# nginx.conf
/etc/nginx

# default
/etc/nginx/sites-enabled

# ehab.html
/var/www/html
```

### Run in the browser
```bash
# <ipAddr>:Port
localhost:81
```
